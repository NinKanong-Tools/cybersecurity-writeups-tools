### What is netdisocver?
Netdiscover is an open-source network reconnaissance tool used to identify live hosts on a network by sending ARP (Address Resolution Protocol) requests and sniffing for replies. It’s particularly useful for network administrators and cybersecurity professionals for tasks like network auditing, device inventory, and identifying unauthorized devices. It operates in both active and passive modes:
      - Active Mode: Sends ARP requests to a specified IP range or network interface to discover live hosts, their IP addresses, MAC addresses, and often the hardware vendor.
      - Passive Mode: Silently sniffs network traffic for ARP packets without sending any requests, making it stealthier but slower, as it relies on existing network activity.

## 4. Active Mode with Interval
````
sudo netdiscover -i eth0 -s 100
````

- `-i eth0` : Specifies the network interface (replace eth0 with your interface, e.g., wlan0).
- `-s 100` : Sets the interval to 100 milliseconds between ARP requests. Adjust this value to control scan speed (smaller values = faster, larger values = slower).

## Usage:
| Option           | Description                   |
| ---------------- | ----------------------------- |
| `-i (interface)` | Specify network interface     |
| `-r (range)`     | Scan a specific subnet (CIDR) |
| `-p`             | Passive mode (listen only)    |
| `-s (ms)`        | Delay between ARP requests    |
| `-S (ms)`        | Scan interval for active scan |
| `-F`             | Show MAC vendor               |
| `-q`             | Quiet mode                    |
| `-P`             | Print results only            |
| `-f (file)`      | Use known hosts file          |
| `-n (number)`    | Number of retries per host    |
| `-d`             | Debug mode                    |



## 9. Limit retries and timeout per host:
````
sudo netdiscover -i eth0 -r 192.168.127.0/24 -n 1 -s 100 -c 1 -f
````

### Why Hacker use netdiscover:
Hackers (and ethical hackers/pentesters) use Netdiscover because it’s a very effective tool for network reconnaissance, especially on local networks.
- Quickly discover live hosts
- Identify MAC addresses and vendors
- Passive scanning for stealth
- First step in an attack chain
- Detect hidden or rogue devices
- Useful in VLAN environments
